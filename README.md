# Матвей и Буран — «НА» и «ДАЙ»

Интерактивная браузерная речевая игра для маленького ребёнка.

## Что находится в архиве

Сохранена первая история с яблоком и добавлены пять новых сценариев — всего шесть:

1. Яблоко туда и обратно.
2. Морковка для Бурана.
3. Буран хочет пить.
4. Чистим Бурана.
5. Игра с мячом.
6. Собираемся в дорогу.

В заданиях чередуются разные действия:

- ребёнок выбирает между словами «НА» и «ДАЙ»;
- нажимает на микрофон и произносит короткую фразу;
- перетаскивает предметы к Бурану;
- выбирает нужный предмет среди нескольких;
- проводит щёткой по боку коня;
- получает похвалу, звёзды и голосовую обратную связь.

## Голосовые задания

Игра не пытается оценивать произношение трёхлетнего ребёнка. Она определяет только наличие голоса. Это сделано специально, чтобы ребёнок не получал ошибку из-за нечёткой речи.

На опубликованном HTTPS-сайте браузер может попросить разрешение на микрофон. Если доступ запрещён или игра открыта напрямую как файл, всегда доступна кнопка «Я сказал!».

Речь героев создаётся встроенным браузерным синтезатором. Игра старается выбрать мужской русский голос. Точный тембр зависит от устройства.

## Запуск

Можно просто открыть `index.html`.

Более стабильный локальный вариант:

```bash
python -m http.server 8000
```

Затем открыть:

```text
http://localhost:8000
```

## Публикация на GitHub Pages

1. Создать новый репозиторий.
2. Загрузить содержимое папки проекта в корень репозитория.
3. Открыть `Settings` → `Pages`.
4. Выбрать `Deploy from a branch`.
5. Выбрать ветку `main` и папку `/root`.
6. Сохранить настройки.

Адрес будет выглядеть примерно так:

```text
https://ИМЯ-ПОЛЬЗОВАТЕЛЯ.github.io/ИМЯ-РЕПОЗИТОРИЯ/
```

Серверная часть и база данных не нужны.
